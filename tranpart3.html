So once you have dug in deeper and learned more about responsive design you are going to be ready to start implementing it.

Responsive design is very powerful but also adds a lot of complexity to development. It may be too complex for each project to start from scratch. So here are some tools you can leverage.

Foundation is a Responsive Framework, that gives you a set of tools for Responsive Design, gives you a flexible grid system, responsive components, sensible defaults for RWD. In fact it is what we used to create the GoResponsive site. It's defiantly a tool that can fast track your development. If we go back to the GoResponsive site, go down to the Developers section and Dive In. You will be taken to Github which is a tremendously powerful collaborative coding platform. We have taken all components that we have built for the GoResponsive site and made them reusable and shareable on Github so you download the project and start experimenting with the project right away. In the project you will find a blank un-styled template and this is a responsive template, so as it gets resized it will turn into the mobile view just like you saw in the GoResponsive site for you to play with. Customize it, and it can form the foundation of your responsive project.

Its alpha version right now but we are working on it and hope to have it production ready very soon. As a matter of fact, Division of Tax Appeals is a site that was ready for a refresh, and we took the Excelsior Web Framework, and took a couple of days to adapt the main pages and you can see it turned very well. We used the adaptive tables, even form layout with a live map, responds just fine.
